# OpenDRIVE XSD Schemas

This directory contains the official ASAM OpenDRIVE V1.7.0 XSD schemas.

## Source

ASAM OpenDRIVE V1.7.0
https://www.asam.net/standards/detail/opendrive/

## Files

- `opendrive_17_core.xsd` - Main schema (includes other schemas)
- `opendrive_17_junction.xsd` - Junction definitions
- `opendrive_17_lane.xsd` - Lane definitions
- `opendrive_17_object.xsd` - Object definitions
- `opendrive_17_railroad.xsd` - Railroad definitions
- `opendrive_17_road.xsd` - Road definitions
- `opendrive_17_signal.xsd` - Signal definitions
